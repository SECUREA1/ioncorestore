# IONCORE Website Package

Fully operational static HTML storefront package built from the uploaded IONCORE product graphics.

## Included
- Homepage
- Complete product marketplace
- Category market pages
- 20 individual product detail pages
- Local demo cart using browser localStorage
- Product modal viewer
- Demo checkout / lead capture form
- All uploaded product images
- CSS and JS assets

## How to run locally
Open `index.html` in a browser.

## How to deploy
Upload the full folder to GitHub Pages, Netlify, Vercel, Render static site, or any web host.

## To make checkout real
Connect the demo checkout to Stripe, Shopify Buy Button, Snipcart, WooCommerce, or a backend API.
